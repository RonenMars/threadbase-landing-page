# Contributing

Thanks for your interest in contributing to this project. It is part of the [Threadbase](https://threadbase.sh/) ecosystem.

## Reporting issues

Open an issue describing what you ran, what you expected, and what happened. Include version (`package.json` or `build.gradle.kts`), OS, and any error output.

## Sending pull requests

1. Open an issue first to discuss the change — keeps PRs focused.
2. Fork the repo, branch off `main`.
3. Make your change. Keep diffs small and focused.
4. Run tests: `npm test`
5. Open a PR against `main`. The PR description should reference the issue.

## Code of Conduct

By participating in this project, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).
