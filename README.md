# threadbase-landing-page

Marketing site for the [Threadbase](https://threadbase.sh/) project. Built with [Next.js](https://nextjs.org/).

## Develop

```sh
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```sh
npm run build
npm start
```
